Face Expression using AI

Download the "face-api.js-master.zip" file , extract it and paste it in your C drive. Copy the path. This is the Face Recognition library.
After you did that, go to the search bar of your computer(You may use 'Run' also) and then type "Environmental Variable" -> Click OK.
Click on "Environmental Variable" button on the appeared dialogue box.
Now there will be two sections divided, each containing path. Click each of them and press Edit.
Click New -> Paste the path of  this "face-api.js-master" file. -> Click OK.

Now just run the index.html...........Enjoy
